# TradeForge 基础设施三阶段总览

---

## 项目简介

TradeForge（锻合）是一个 AI 原生的跨境工业品自动化撮合平台，核心引擎为 15 节点智能决策状态机，覆盖从买家询盘到订单履约的全自动化闭环。内置出口合规审查、多轮 AI 谈判、背靠背套利对冲、防篡改文档生成等金融级能力。

平台采用 **混合云架构**：香港/新加坡云端承载业务系统，本地 GPU 工作站集群运行 AI 推理，通过 WireGuard VPN 加密互通。商业敏感数据（报价策略、客户画像、合规判断）始终保留在本地，不经过任何第三方。

## 核心设计原则

| 原则 | 说明 |
|------|------|
| **渐进式投入** | 每阶段硬件全部复用，新增投入仅为增量部分，避免重复采购 |
| **数据主权** | 商业敏感推理（报价、画像、合规）始终在本地 GPU 运行，不经过第三方 |
| **云端借力** | 非敏感的寻源数据清洗交给低成本 API（DeepSeek），用月费替代硬件 |
| **多重冗余** | 云端负载均衡 + 本地工作站互备 + 多供应商 API + UPS 防断电 |
| **弹性扩展** | 系统架构天然支持横向扩展：无状态应用节点、分布式事件总线、读写分离数据库 |
